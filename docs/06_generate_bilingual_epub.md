# Generate Bilingual EPUB

Bilingual output is planned for a later phase.

The intended layout modes are:

- Japanese above Chinese
- Chinese below Japanese
- Side-by-side
- Paragraph-by-paragraph

## MVP Behavior

The current MVP keeps the interface and option names ready, but does not perform translation. Translation should only run when the user configures a local translation engine or a user-provided API key.

## Alignment Guidance

Prefer paragraph alignment first. Sentence-level alignment often breaks literary prose and creates noisy layouts.
