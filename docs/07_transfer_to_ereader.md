# Transfer To E-reader

Common transfer methods:

- Vendor WiFi transfer page
- USB file copy
- Syncthing
- Local network share

## Hanvon-style WiFi Transfer SOP

1. Open the device's WiFi or local transfer tool.
2. On your computer, open the transfer URL shown by the device.
3. Upload the generated EPUB.
4. In KOReader, set the root folder to the transfer directory.
5. Refresh KOReader's file browser.
6. Open the converted EPUB.

This is often smoother than using the built-in reader's library import flow.
