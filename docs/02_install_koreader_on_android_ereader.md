# Install KOReader On Android E-ink Readers

Use this when your device can install Android APK files.

## Suitable Devices

- Hanvon Android e-ink readers
- BOOX / Onyx Android e-ink readers
- iReader-style Android devices that allow APK installation
- Other Android-based e-ink readers with local file access

## Not Suitable

- Fully closed systems that cannot install APK files
- Devices where the vendor blocks unknown-source installation

## Steps

1. Open the official KOReader website or GitHub Releases page on your computer.
2. Download the Android APK that matches your device CPU.
   - Most older or compatibility-oriented devices can try the ARM build first.
   - ARM64 may work on newer devices.
3. Copy the APK to the e-ink reader.
4. On the device, allow installing apps from unknown sources.
5. Install KOReader.
6. Open KOReader once and grant file access if Android asks.
7. Set the starting folder to your book transfer folder.

Official sources:

- https://koreader.rocks/
- https://github.com/koreader/koreader/releases

Avoid random APK mirrors unless the official sources are unavailable.
