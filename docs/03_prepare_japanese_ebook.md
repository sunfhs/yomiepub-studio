# Prepare A Japanese Ebook

Supported MVP input formats:

- `.epub`
- `.txt`
- `.html`
- `.xhtml`

## Legal Requirement

Use files you legally own or are allowed to process. This tool does not provide download, search, DRM removal, or piracy features.

## Recommended Source Quality

Prefer files with:

- UTF-8 text
- Clean chapter structure
- Japanese language metadata when available
- Minimal DRM-free EPUB packaging

Avoid files that are image-only scans if you need furigana generation. Image-only books require OCR first, which is outside the MVP.
